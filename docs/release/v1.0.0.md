### Release v1.0.0

#### Resumen

`v1.0.0` consolida SentinelOps como un producto de laboratorio reproducible para DevSecOps, acceso remoto seguro, policy as code, observabilidad, persistencia y medición de rendimiento.

Esta versión toma el historial previo del proyecto, incluyendo versiones internas como `2.4.1` y variantes asociadas a `commit1-SentinelOps`, y lo ordena como una línea estable por fases.

#### Fuera de alcance

Chaos engineering queda reservado para una versión posterior. No se incluye en `v1.0.0` para mantener el release final estable y reproducible.
